PecoRename

Introduction

PecoRename est un utilitaire permettant de renommer simultanément plusieurs fichiers qui doivent être renommés de la même manière.
Divers techniques de renommage sont disponibles : ajouter une extension, numéroter les fichiers, remplacer...

Utilisation du logiciel

L'utilisation de PecoRename est très simple. La plupart des fonctionnalités sont intuitives. Pour cette raison, le menu ne sera pas expliqué en détail - les noms sont explicites et reprennent les intitulés des boutons.

La fenêtre est divisée en deux parties :

  • la liste des fichiers en haut et
  • les fonctions de renommage en bas.

La liste des fichiers sélectionnés

Cette liste contient les fichiers et répertoires à renommer.
PecoRename ne fait pas de différence entre les fichiers, les liens et les répertoires - seules les périphériques sont ignorés. D'autre part, seuls les fichiers d'un même répertoire peuvent être sélectionnés.

Pour sélectionner les fichiers et répertoires, vous avez deux possibilités :

  • Vous pouvez choisir les fichiers en cliquant sur le bouton "Selectionner..." en haut à gauche.
    Attention: Malheureusement, la fenêtre de sélection de fichier de BeOS Release 5 (PE) a un bug qui fait crasher BeOS quand plus de 120 fichiers sont sélectionnés et ouverts !
  • Vous pouvez aussi - très facilement - sélectionner vos fichiers via le Tracker et les faire glisser dans la fenêtre de PecoRename.
Dès qu'il y a des fichiers sélectionnés, le répertoire courant s'affiche au dessus de la liste. Les fichiers peuvent être triés par nom, taille et date de dernière modification. Pour cela, il suffit de cliquer sur le titre de la colonne correspondant au critère de tri souhaité.

La colonne "Apercu" affiche pourchaque fichier son nouveau nom après renommage. Seuls les fichiers à renommer apparaissent dans la liste. Si un conflit est détecté à cause d'un doublon dans les noms de fichiers (noms après renommages et noms des autres fichiers non renommés du répertoire), les fichiers concernés apparaissent en rouge.

Les fonctions de renommage

Quand tous les fichiers voulus sont sélectionnés, vous pouvez les renommer.
Le renommage se fait selon les différents modes suivants:

  • Rechercher / Remplacer

    Cette option permet de rechercher une chaîne de caratères dans les noms des fichiers et de la remplacer par une autre.
    Voici les paramètres possibles:
    • Différentiation MAJ/min: Indique, quand la case est cochée, qu'on souhaite différencier les majuscules et les minuscules.
    • Expression régulière: Vous pouvez Rechercher/Remplacer des expressions régulières.
      Sans entrer dans le détail des expressions régulières, il faut savoir que la chaîne recherchée est reprise dans la chaîne de remplacement gr√¢ce à '\$'.
    • Rechercher: La chaîne de caratères recherchée
    • Remplacer par: La chaîne de remplacement

  • Insertion / Remplacement

    Vous pouvez insérer du texte à une certaine position dans le nom de(s) fichier(s).
    Voici les différentes options :
    • Insert or replace: Le texte doit-il être inséré ou remplace-t-il le texte existant à la position demandée?
    • Texte: Quel est le texte qui doit être mis à cette position?
    • Position: A quelle position?
    • Direction: Vous pouvez choisir d'indiquer la position en partant du début (à gauche) ou de la fin (à droite) du nom des(s) fichier(s).
      Exemple: La position "3" "à partir de la fin" signifie que le texte sera inséré 3 caractères avant la fin du nom de chaque fichier.

  • Numérotation

    Les fichiers seront numérotées de facon croissante. L'ordre dans lequel sont triés les fichiers sources (par Nom/Taille/Modifié le) détermine l'ordre de la numérotation.
    • Format: Vous pouvez choisir ici le nombre de positions pour la numérotation (de 1 à 4 positions).
    • Commencer par: Indique le premier nombre utilisé pour la numérotation (par défaut, la numérotation commence à 0) .
    • Texte en tête, Texte en fin: Vous pouvez indiquer le texte qui sera inséré avant ou après le numéro du fichier.

  • Ajouter une extension

    Cette option permet de trouver une extension adéquate à un fichier.

    Cette fonction ne fournit pas toujours le résultat attendu. C'est pourquoi, il est nécessaire de comprendre la manière dont cette fonction est réalisée :

    Dans BeOS - comme sur internet - à chaque fichier est attribué un type appelé type MIME. Si vous importez des fichiers d'un environnement autre que BeOS, ces fichiers doivent être identifiés. Pour cela, BeOS utilise divers méthodes de reconnaissance et, en dernier lieu, utilise l'extension du fichier.

    PecoRename réalise le travail inverse : pour un fichier donné et son type MIME, PecoRename recherche l'extension appropriée - telle qu'elle est définie dans Preferences/FileTypes.

    Pour que cette fonction PecoRename ait un comportement correct, deux conditions doivent être remplies :

    1. Le fichier doit être identifié. Si vous n'êtes pas certain que le fichier a été identifié par BeOS, vous pouvez provoquer cette indentification en sélectionnant le fichier dans le Tracker et en choisissant File/Identify (ou Identify dans le menu contextuel du bouton droit de la souris).
    2. Une extension par defaut doit être attribué pour ce type de fichier
      Pour cela, choissez "File Types" dans le menu des préférences. Cette application vous permet d'jouter une extension pour chaque type de fichier.
      PecoRename utilise toujours la première extension trouvée. Vous aurez donc souvent une extension non souhaitée (".jpeg" au lieu de ".jpg" ou ".mp2" au lieu de ".mp3", mais avec l'option "Rechercher/Remplacer" de PecoRename, vous avez l'outil ideal pour corriger cela.

    Les options suivantes sont disponibles :

    • Remplacer l'ancienne extension: Permet de choisir entre l'ajout ou le remplacement de l'extension.
    • Majuscule / Minuscule: Permet d'écrire la nouvelle extension en majuscule ou en minuscule.

Majuscules / Minuscules

Comme son nom l'indique, cette fonction convertit les lettres des noms de fichiers en majuscules ou en minuscules.

C'est parti !

Losque tous les réglages du renommages sont faits, il suffit de cliquer sur le bouton "C'est parti !" et toute les fichiers seront renommé selon l'apercu.
