PecoRename

Einführung

PecoRename ist ein Utility, das dabei hilft, mehrere Dateien umzubenennen, die nach einem ähnlichen Schema umbenannt werden sollen.
Dabei hat man verschiedene Möglichkeiten: Mit einer Endung oder einem Präfix versehen, Nummerieren, Suchen&Ersetzen...

Bedienung des Programms

PecoRename ist eigentlich sehr einfach zu bedienen. Die meisten Funktionen sollten sich von selbst erklären. Deshalb gehe ich auch nicht weiter auf das Menü ein - die Bezeichnungen sprechen für sich bzw. finden sich in Buttons wieder.

Das Fenster selbst unterteilt sich grob in zwei Bereiche:

  • die Dateiansicht im oberen Bereich und
  • die Umbenennungsfunktion im unteren Teil.

Die Dateiansicht

In der Dateiansicht werden die umzubenennenden Dateien und Verzeichnisse angezeigt.
PecoRename macht keinen Unterschied zwischen Dateien, Links und Verzeichnissen - nur Laufwerke werden ignoriert. Ausserdem können nur Dateien eines Verzeichnisses gewählt werden.

Um Dateien in PecoRename zu bringen, hat man zwei Möglichkeiten:

  • zum Einen kann man die Dateien durch ein FilePanel (Dateidialog) wählen. Dieses öffnet sich, wenn man auf den Button "Bitte wählen" (links oben) klickt.
    Achtung: Leider hat das FilePanel in der Release 5 einen Bug, so dass das gesamte BeOS abstürzt, wenn man mehr als ca. 120 Dateien anwählt und öffnet!
  • Die Zweite - und bequemere - Möglichkeit: Man selektiert die gewünschten Dateien im Tracker und zieht sie in das PecoRename-Fenster. Um Dateien wieder aus der Dateiansicht zu entfernen kann man sie genauso einfach auch wieder herausziehen!
Befinden sich Dateien in der Liste, so wird deren gemeinsamer Pfad über der Liste angezeigt. Die Dateien können nach Name, Größe und Änderungsdatum sortiert werden. Dazu klickt man einfach auf den entsprechenden Listentitel.

In der Spalte "Vorschau" sieht man, welche Dateien wie umbenannt werden. Nur Dateien, an denen Veränderungen vorgenommen werden, werden angezeigt. Falls es dabei durch doppelte Namen zu Konflikten kommen würde, werden die betreffenden Dateien rot hervorgehoben.

Die Umbenennungsfunktionen

Hat man die gewünschten Dateien in der Dateiansicht, geht es darum, sie umzubenennen.
Dazu gibt es folgende Funktionen (Modi):

  • Suchen und ersetzen

    Wie der Name schon sagt, werden hier in den Namen Zeichenketten gesucht und ersetzt.
    Dabei kann man folgende Parameter einstellen:
    • Groß/Kleinschreibung beachten: Wenn dieses Feld markiert ist, wird bei der Suche die Groß/Kleinschreibung beachtet.
    • Regulärer Ausdruck (regular expression): Man kann auch mit Hilfe von sog. regulären Ausdrücken - d.h. Suchmustern - suchen.
      Es würde zu weit führen, diese hier zu beschreiben. Wichtig ist für diejenigen, die sich damit auskennen, dass der gefundene Suchstring im Ergebnisstring durch das Token '\$' eingesetzt werden kann.
    • Suchmuster: Wonach wird gesucht?
    • Ersetzen durch: Wodurch soll ersetzt werden?

  • Einfügen / Überschreiben

    Man kann an einer bestimmten Stelle im Dateinamen einen Text einsetzen.
    Hierbei gibt es folgende Optionen:
    • Einfügen oder überschreiben: Soll der Text eingefügt werden oder den bestehenden Text ab dieser Position überschreiben?
    • Text: Welcher Text soll eingesetzt werden?
    • Position: An welcher Stelle?
    • Richtung: Man kann bestimmen, ob von vorne oder hinten die n-te Position gemeint ist.
      Beispiel: Eine Position "3" und eine Richtung "von hinten" bedeutet, dass der Text aber der 3. Stelle von hinten eingesetzt wird.

  • Entfernen

    Es werden Buchstaben aus dem Namen entfernt. Dazu stellt man einfach ein, von wo bis wo und damit ist die Sache auch schon erledigt! :-)

  • Nummerieren

    Die Dateien werden der Reihe nach durchnummeriert. Entscheidend ist dabei auch die Sortierung (nach Name/Größe/Änderungsdatum) in der Dateiansicht.
    • Format: Hier wählt man, auf wieviele Stellen die Zahl fixiert ist.
    • Starte mit: Es kann eine Startzahl angegeben werden, mit der die Nummerierung beginnt.
    • Text davor, Text danach: Hier kann man Text angeben, der vor bzw. nach der Zahl eingesetzt wird.

  • Dateiendung hinzufügen

    Es wird versucht, die richtige Dateiendung zu finden, und diese wird dann an den Dateinamen angehängt.

    Allerdings wird dies nicht immer wunschgemäß funktionieren. Daher ist es sinnvoll, die Funktionsweise zu verstehen:

    Im BeOS wird jede Datei - wie im Internet - durch einen sog. MIME-Typen gekennzeichnet. Wenn man nun Dateien von einer Nicht-BeOS-Umgebung bekommt, müssen diese irgendwie erkannt werden. Dazu verwendet das BeOS verschiedene Erkennungsmechanismen und zuletzt - falls eine Datei anders nicht erkannt werden kann - die Dateiendung.

    PecoRename geht den umgekehrten Weg: Zu einem Dateitypen wird die entsprechende Endung gesucht - so, wie sie in Preferences/FileTypes eingestellt ist.

    Damit also PecoRename seine Sache gut macht, müssen zwei Voraussetzungen gegeben sein:

    1. Die Datei muss identifiziert sein.
      Wenn man sich nicht sicher ist, ob das BeOS eine Datei schon erkannt hat, kann man dies im Tracker machen: Datei anwählen und im Menü File/Identify wählen. Dies funktioniert nur auf BFS-Datenträgern, nicht mit z.B. FAT32 (Windows)!
    2. Es muss eine Dateiendung für eine Datei dieses Typs gespeichert sein.
      In Preferences/FileTypes lassen sich alle möglichen Dateitypen bearbeiten. Dort kann man auch die Endung einstellen.
      PecoRename verwendet immer die erste, gefundene Endung. Hierdurch wird man oft eine nicht ganz wunschgemäße Endungen wie ".jpeg" (statt ".jpg") oder ".mp2" (statt ".mp3") bekommen, aber man hat mit PecoRename (Suchen&ersetzen) das Werkzeug in der Hand, um diese Endungen zu korrigieren!

    Es gibt folgende Optionen:

    • Alte Endung ersetzen: Soll die Erweiterung angehängt werden, oder eine evtl. vorhandene alte Erweiterung ersetzen?
    • Groß/Kleinschreibung: Wie soll die Endung geschrieben werden (gross/klein/wie voreingestellt)?

Los geht's!

Wenn alle Einstellungen vorgenommen wurden, braucht man nur noch auf den "Los geht's"-Button zu klicken und alle Dateien werden entsprechend der Vorschau umbenannt!
