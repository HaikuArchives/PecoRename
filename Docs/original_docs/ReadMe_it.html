PecoRename

Introduzione

PecoRename è una utility che consente di rinominare molti files secondo un pattern predefinito.
Consente di rinominare i files in molti modi: aggiungendo un'estensione, numerando i files in ordine, cercando e sostituendo stringhe, e molto altro...

Uso del software

PecoRename è molto facile da usare. Molte funzioni sono molto lineari. I nome delle funzioni dei menu sono autoesplicanti e tutti i pulsanti sono chiaramente descritti.

La finestra si divide in due aree:

  • l'elenco dei file nell'area superiore e
  • le funzioni di ridenominazione.

L'elenco dei files

Questo elenco mostra i files e le cartelle che saranno rinominate.
PecoRename non fa differenza fra files, links e cartelle - solo i devices sono ignorati. Solo files che risiedono nella stessa cartella possono essere selezionati.

Ci sono due modi per scegliere i file da rinominare:

  • Potete selezionare i files con un file panel cliccando sul pulsante "Seleziona..." in alto a sinistra.
    Attenzione: Purtroppo il file panel standard di BeOS Release 5 (PE) ha un bug che causa un crash quando vengono selezionati più di 120 files!
  • La seconda - e più comoda - possibilità: Selezionate i files desiderati direttamente dal Tracker e trascinateli nella finestra di PecoRename. Per rimuovere alcuni files dall'elenco, basta trascinarli via!
Quando ci sono files in elenco, il percorso comune verrà mostrato sulla lista. I files possono essere ordinati per nome, dimensioni e in base alla data dell'ultima modifica. Per fare ciò basta cliccare sul relativo pulsante in testa all'elenco.

Nella colonna "Anteprima" potete vedere un'anteprima dei files con i relativi nuovi nomi. Solo i files da modificare sono mostrati. Se emergeranno conflitti dovuti a nomi che si ripetono, i relativi files verranno marcati in rosso.

Le funzioni di ridenominazione

Quando i files desiderati sono elencati, è possibile rinominarli.
Per loro sono disponibili le seguenti funzioni(modi):

  • Cerca e sostituisci

    Come dice il nome, questa funzione cerca e sostituisce alcune parti nalla stringa dei nomi.
    Potete regolare i seguenti parametri:
    • Considera Maiusole/minuscole: Se scegliete questa opzione, le maiuscole e le minuscole non verranno trattate alla stessa maniera.
    • Espressione regolare : Potete anche cercare e sostituire le stringhe usando espressioni regolari.
      La descrizione delle espressioni regolari va al di la degli scopi di questo manuale. Per coloro i quali desiderano sapere come lavorare con le espressioni regolari, la stringa trovata può essere inserita nella stringa risultante con il token '\$'.
    • Trova pattern: Cosa state cercando?
    • Sostituisci con: Con cosa andrebbe sostituito?

  • Rimuovi

    Questa funzione rimuove alcuni caratteri dai nomi. Inserite semplicemente il range ed è fatta! :-)

  • Inserisci / sostituisci

    Potete inserire un testo in una certa posizione del nome del file.
    Sono valide le seguenti opzioni:
    • Inserisci o sostituisci: Il testo dovrà essere inserito o dovrà sostituire il testo presente da questa posizione?
    • Testo: Che testo inserire?
    • Posizione: In quale posizione?
    • Direzione: Potete decidere se inserire nella n-esima posizione a partire dalla testa o dalla fine del testo.
      Esempio: Una posizione "3" e una direzione "dalla fine" vuol dire che il testo sarà inserito partendo dalla terza posizione a partie dalla fine.

  • Numerazione

    I files verranno numerati in ordine. I files sono numerati a seconda dell'ordine dell'elenco (in base a nome/dimensione/data dell'ultima modifica).
    • Formato: Scegliete quanti posti occupa il numero.
    • Comincia con: Indicate con quale numero si desidera partire.
    • Testo prima, Testo dopo: Potete specificare se il testo sarà aggiunto prima o dopo il numero.

  • Aggiungi un'estensione

    PecoRename può trovare la giusta estensione da aggiungere al file.

    Non sempre questa funzione darà i risultati attesti. Perciò è necessario capire come funziona:

    In BeOS - come in Internet - ogni file viene distinto da un cosiddetto tipo MIME. Se ottenete un file da un ambiente estraneo a BeOS il file deve essere riconosciuto. BeOS per far ciò usa diversi meccanismi tra cui - se il file non può essere riconosciuto in altro modo - l'interpretazione della sua estensione.

    PecoRename lavora al contrario: per un dato file viene cercata la relativa estensione - come stabilito in Preferences/File Types.

    Affinché PecoRename lavori bene bisogna devono verificarsi due ipotesi:

    1. Il file deve essere identificato.
      Se non siete sicuri che BeOS abbia già riconosciuto il file, potet farlo tramite Tracker: Selezionate il file e scegliete File/Identify dal menu. Questo funziona solo se il file si trova su un disco BFS (non funziona con dischi FAT, HFS o altri filesystem).
    2. Deve esistere un'estensione per un file di quel tipo
      In Preferences/File Types possono essere aggiustati tutti i tipi di file. Potete anche regolare l'estensione del file.
      PecoRename usa sempre la prima estensione trovata. Per questo motivo, avrete un'estensione come ".jpeg" (invece di ".jpg") o ".mp2" (invece di ".mp3") - con PecoRename (usate Cerca e sostituisci) potrete correggerlo!

    Sono valide le seguenti opzioni:

    • Sostituisci la vecchia estensione: L'estensione sarà accodata o sostituirà la vecchia, se necessario?
    • Maiuscolo / minuscolo: Come scriveremo l'estensione (maiuscolo/minuscolo/default)?

  • Maiuscolo / minuscolo

    Questa funzione è ovvia! Converte tutte le lettere in minuscolo o in maiuscolo.

Si parte!

Quando tutto è impostato, dovete solo premere il pulsante "Si parte!" e tutti i file saranno rinominati secondo l'anteprima!

Strumenti

Nel menu Strumenti avete la possibilità di creare uno script per la shell secondo le impostazioni selezionate. Questo script può essere lanciato con un doppio-click e funziona esattamente come un click sul pulsante "Si parte!". Potrete creare gli script una volta sola ed eseguirli ancora e ancora - anche senza PecoRename!
