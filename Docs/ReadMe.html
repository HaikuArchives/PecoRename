PecoRename

Introduction

PecoRename is a utility which allows you to rename many files according to a preset pattern.
It lets you rename your files in many different ways: Add an extension, Number your files in order, Search and replace, and much more...

Usage of the program

PecoRename is very easy to use. Most functions are very straightforward. The names of the menu functions speak for themselves and all buttons are clearly labeled.

The window can be divided into two areas:

  • the file view in the top area and
  • the renaming function in the bottom part.

The file view

The file view shows the files and directories which will be renamed.
PecoRename doesn't make a difference between files, links and directories - only devices are ignored. Only files in the same directory can be chosen.

There are two ways to choose which files to rename:

  • You can choose the files with a file panel by clicking on the "Please select..." button on the top left.
    Attention: Unfortunately the file panel in BeOS Release 5 (PE) has a bug that causes BeOS to crash when more than about 120 files are chosen and opened!
  • The second - and more comfortable - possibility: You can select the files you want in the Tracker and drag them into the PecoRename window. To remove files from the file view, you can drag them out just as easy!
When there are files in the list, the common path will be shown over the list. The files can be sorted by name, size and modification time. To do this simply click onto the referring list title.

In the column "Preview" you can see which files will be renamed into what names. Only the files to be modified are shown. If conflicts will occur because of double file names, the referring files will be marked in red.

The renaming functions

When you have the files you wish in the file view, it's time to rename them.
For that there are the following functions (modes):

  • Search and replace

    As the name states, this function searches and replaces parts of the names.
    You can adjust the following parameters:
    • Case sensitive: If you mark this field, upper and lower case will be paid attention to.
    • Regular expression: You also can search&replace using regular expressions.
      It would be to much to describe regular expressions here. For those who know how to work with regular expressions, the found string can be inserted in the result string with the token '\$'.
    • Find pattern: What are you looking for?
    • Replace with: What should it be replaced with?

  • Remove

    This function removes characters from the name. You simply have to enter the range - and that was it! :-)

  • Insert / replace

    You can put in text at a certain position of the filename.
    Here exist the following options:
    • Insert or replace: Should the text be inserted or overwrite the existing text starting with this position?
    • Text: Which Text shall be put in?
    • Position: At which position?
    • Direction: You can decide if you mean the n-th position from the left or from the right.
      Example: A position "3" and a direction "from the back" mean, that the text will be put in starting with the 3rd position from the back.

  • Numbering

    The files will be numbered in order. Files are numbered depending on the order (by name/size/modification time) of the file view.
    • Format: Choose how many places the number is fixed to.
    • Start with: Here you can indicate which number to begin numbering with.
    • Text before, Text behind: You can specify whether text will be added before or behind the number.

  • Add file extension

    PecoRename can try to find the correct file extension to be appended to the filename.

    This will not always work as desired. Therefore it is important to understand how it works:

    In the BeOS - like in the Internet - every file is marked by a so-called MIME type. If you get a file from a non-BeOS environment the new file's MIME type must be recognized. For this the BeOS uses different recognition mechanisms and as a last resort - if the file can not be recognized in another way - the file extension.

    PecoRename goes the reverse way: For a given file type the referring extension will be searched - like it is set in Preferences/File Types.

    For PecoRename to do a good job of this, two assumptions must be given:

    1. The file must be identified.
      If you are not sure if the BeOS already has recognized a file, you can do this in the Tracker: Select the file and select File/Identify from the Menu. This only works on BFS formatted devices. (will not work on FAT or HFS or any other filesystem type)
    2. There must be an extension set for a file of that type
      In Preferences/File Types all kinds of file types can be adjusted. You can also set the file extension.
      PecoRename always uses the first extension found. Because of this, you will often have a non-desired extension like ".jpeg" (instead of ".jpg") or ".mp2" (instead of ".mp3") - but with PecoRename (use Search & Replace) you have the tool to correct it!

    There are the following options:

    • Replace old extension: Should the extension be appended or should it replace a old one, if necessary?
    • Uppercase / lowercase: How should the extension be written (upper/lower case/default)?

  • Uppercase / lowercase

    This function should be obvious! It converts all the letters of the filename to uppercase or lowercase.

Let's do it!

When all settings are made, you only have to hit the "Let's do it!" Button and all files will be renamed according the preview!
